# Asset checklist — ภาพที่ต้องเตรียม (สไตล์ chibi isometric ตามรูปอ้างอิง)

วางไฟล์ตามโฟลเดอร์ด้านล่าง เป็น **PNG พื้นหลังโปร่งใส** เลือกได้ 2 แนวทาง:

- **แนวทาง A — ง่าย/เร็ว (แนะนำเริ่มต้น):** ใช้ "ฉากเต็ม" 1 รูป เป็นพื้นหลัง + ตัดเฉพาะ "ตัวละคร" มาวางทับให้ขยับได้ → เตรียมแค่กลุ่ม ★ ก็พอ
- **แนวทาง B — ยืดหยุ่น:** แยกทุกชิ้น (เฟอร์นิเจอร์/ต้นไม้/ป้าย) เพื่อจัดวาง/เพิ่ม-ลดเองได้ → เตรียมกลุ่ม ◆ ด้วย

> ทุกชิ้นใช้ **มุม isometric เดียวกัน + ทิศแสงเดียวกัน** (ตามรูป) จะ composite ได้เนียน

---

## background/  (ฉาก)
- ★ `scene-full.png` — **ฉากเต็มตามรูปอ้างอิง** (ดาดฟ้า+เฟอร์นิเจอร์+เมือง แต่ **ไม่มีตัวละคร**) ≈ 1920×1080
  _ถ้าใช้แนวทาง A เตรียมแค่ไฟล์นี้พอ (เอาตัวละครออกจากรูป)_
- (แนวทาง B) `city.png` — สกายไลน์+ฟ้า (พื้นหลังไกล), `rooftop.png` — ดาดฟ้าเปล่า (พื้นไม้/กระเบื้อง+ขอบ+บันได+ห้องบันได+แอร์+ไฟราว) โปร่งใส

## characters/  (★ แยกไฟล์ละตัว, ตัดพื้นหลังออก โปร่งใส)
ของเราใช้ 3 ตัวหลัก: pixel (UX/UI เสื้อขาว+แว่น), cora (Team Lead), milo (Coordinator เสื้อแดง ผมหยิก)

**ไม่ต้องทำครบ 4 ทิศ × หลายสีหน้า** — โค้ดเติม flip/เด้ง/zzz/จุดคิดให้เอง เลือกระดับได้:

- ★ **ขั้นต่ำ (1 รูป/ตัว):** `pixel-idle.png` `cora-idle.png` `milo-idle.png` — ท่ายืน 3/4 หน้า (รูปหลัก)
- ◆ **แนะนำ (3 รูป/ตัว = ได้สีหน้าตามอารมณ์):**
  - `*-idle.png` (หน้าปกติ) → ยืน/เดิน/ทำงาน/หลับ
  - `*-think.png` (หน้าครุ่นคิด) → ตอน AI กำลัง generate
  - `*-happy.png` (ยิ้ม) → ตอนงานผ่าน/อนุมัติ
- ○ **เต็มสุด (ทีหลังได้):** เดินหันจริง — เหลือ 3 ทิศ (ซ้าย/ขวา flip เอา): `*-down-1/2.png` `*-up-1/2.png` `*-side-1/2.png`  หรือ sprite sheet `*-sheet.png`
- (ออปชัน panel) หน้าตรงครึ่งตัว: `*-portrait.png`

> เคล็ดลับให้ AI generate ให้สม่ำเสมอ: ใช้ prompt/ตัวละครเดียวกัน เปลี่ยนแค่ "สีหน้า/ท่า" และ render บนพื้นโปร่ง/พื้นเดียวกัน ทุกไฟล์ขนาด+มุมเท่ากัน

## สเปกไฟล์ (สำคัญ)
| หัวข้อ | ค่าแนะนำ |
|---|---|
| ฟอร์แมต | PNG-32 **พื้นหลังโปร่งใส** (ตัวละคร/พร็อพ) · scene-full/city เป็น PNG/JPG ก็ได้ |
| ความละเอียด | ทำ **@2x** เผื่อจอ retina (เช่น ตัวละครสูง ~400px ไฟล์, แสดงจริง ~200px) |
| มุมมอง | isometric มุมเดียวกันทุกชิ้น (≈ มุมเดียวกับรูปอ้างอิง) |
| แสง/เงา | ทิศแสงเดียวกัน · เงาใต้ตัวละคร/พร็อพ แยกไฟล์หรือ baked จาง ๆ ก็ได้ |
| ตั้งชื่อ | ตัวพิมพ์เล็ก-ขีดกลาง (`desk-imac.png`) ตามลิสต์ด้านบน |

## เริ่มเร็วสุด (ขั้นต่ำ 4 ไฟล์)
`scene-full.png` + `pixel.png` + `cora.png` + `milo.png` → พอให้ขึ้น monitor หน้าตาแบบนี้ได้เลย (ตัวละครขยับ/เปลี่ยนสถานะได้)
